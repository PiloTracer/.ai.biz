# BIZ-06: Operations

## Questions to explore

1. What is your delivery process? (Intake → Build → Review → Ship)
2. Where are the bottlenecks? (Communication, scope creep, quality)
3. What can you standardize? (Templates, checklists, tools)
4. What can you delegate? (Admin, QA, deployment)
5. How do you handle scope changes? (Change order process)

## Output

- Delivery workflow map
- Known bottlenecks with fixes
- Standardization opportunities
- Delegation plan
- Change order process
